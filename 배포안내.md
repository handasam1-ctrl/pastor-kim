# 김일수 목사 포털 — GitHub Pages 배포 안내

## 이 폴더에 있는 파일들

| 파일 | 역할 |
|------|------|
| `index.html` | 웹사이트 본체 |
| `manifest.json` | 앱 이름·아이콘·색상 정의 |
| `sw.js` | 오프라인 캐싱 (Service Worker) |
| `icon-192.png` | 앱 아이콘 (작은 것) |
| `icon-512.png` | 앱 아이콘 (큰 것) |

---

## GitHub Pages 배포 방법 (순서대로)

### 1단계 — GitHub 가입
- [github.com](https://github.com) 접속 → Sign up
- 아이디 예: `pastor-kimils` (영문, 숫자, 하이픈만 가능)

### 2단계 — 새 저장소 만들기
- 로그인 후 우측 상단 **＋** 버튼 → New repository
- Repository name: `pastor-kim` (영문)
- Public 선택 (무료 배포 필수)
- **Create repository** 클릭

### 3단계 — 파일 올리기
- 저장소 페이지에서 **"uploading an existing file"** 클릭
- 이 폴더의 파일 **5개 모두** 드래그 앤 드롭
  - index.html
  - manifest.json
  - sw.js
  - icon-192.png
  - icon-512.png
- 아래 **Commit changes** 클릭

### 4단계 — Pages 활성화
- 저장소 상단 **Settings** 탭 클릭
- 왼쪽 메뉴에서 **Pages** 클릭
- Source: **Deploy from a branch**
- Branch: **main** 선택 → **Save**
- 1~2분 후 주소 생성됨

### 5단계 — 완료!
주소 형식: `https://아이디.github.io/pastor-kim`

예: `https://pastor-kimils.github.io/pastor-kim`

## 아이폰에서 앱처럼 사용하는 방법

1. 위 주소를 **사파리(Safari)** 로 엽니다
   (크롬으로 열면 홈 화면 추가 안 됨)
2. 하단 가운데 **공유 버튼** (□ 위에 화살표) 탭
3. **"홈 화면에 추가"** 탭
4. 이름 확인 후 **추가** 탭
5. 홈 화면에 앱 아이콘이 생깁니다 ✓
